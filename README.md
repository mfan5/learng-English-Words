# 开口说物

一个面向英语口语学习者的生活物品词汇网站。

用户可以按厨房、客厅、卧室、浴室、办公室和街区六个生活场景学习常见物品。每张词卡包含真实图片、中英对照、音标、发音、常见搭配和生活口语例句，并可记录当天已经学会的卡片。

图片将以压缩后的常规网页尺寸保存在项目本地，避免依赖外部图片服务。
